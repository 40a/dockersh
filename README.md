dockersh
========

A shell which places uses into individual docker containers

Configuration
=============

~/.dockershrc.json
------------------

Local (per user) settings for a specific user's dockersh instance.




/etc/dockershrc.json
--------------------

Global settings for all dockersh instances. Allows you to disable settings
in ~/.dockershrc
